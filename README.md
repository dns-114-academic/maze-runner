# Maze Runner — Pathfinding & Genetic Algorithms

**Course project** | école d'ingénieurs — Oct. 2025–Jan. 2026 | Python

## Overview

Maze generation and solving combining classical graph algorithms with an evolutionary approach.

## Algorithms

### Maze Generation

- **DFS** (Depth-First Search) — randomised recursive maze generation

### Classical Solver

- **Dijkstra's algorithm** — shortest path from entry to exit

### Genetic Algorithm Solver

- Population of ~100 candidate paths
- **Fitness function** — based on distance to exit and cells explored
- **Crossover** — path segment recombination
- **Mutation** — random direction changes
- **Pheromone escape mechanism** — guides exploration away from dead ends
- Convergence visualisation over generations

## Visualisation

- Animated maze generation
- Step-by-step Dijkstra solving
- Generation-by-generation GA convergence

## Stack

`Python` · `matplotlib` · `numpy`
